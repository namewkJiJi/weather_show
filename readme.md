# 天气信息展示系统

这是一个天气信息展示系统，用户可以从该系统查看实时天气，未来天气预报，过往天气情况以及灾害预警信息。

## 程序环境

- 后端：python3 + flask框架
- 前端：html，css，js原生代码
- 数据库：mysql
- 数据来源：和风天气api，天气网

## 程序设计



### 后端：

**BluePrint**：采用flask的blueprint，将web网站分成不同的独立模块，便于独立开发，后期维护与功能更新

**ORM**：利用对象操作关系数据库的表，避免书写原生sql带来的不便性



### 前端

**js动态渲染页面**：动态加载图表数据等信息。更新数据时不需要刷新整个网页，增加渲染速度。
